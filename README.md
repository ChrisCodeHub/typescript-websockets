
initialise and setup tooling

```bash
npm init -y

npm install -D typescript @types/node @types/express ts-node nodemon
npm install express
npm install lodash
npm install ws
npm install -D @types/lodash @types/ws

```


transpile and run
```bash
npm run build
node ./dist/server.js
```
